# 基于FPGA的低光照增强算法

## 部分文件说明

- `img`：包含所有的实验结果截图
- `LLEHDL`：包含Simulink模型和Quartus工程
  - `hdlsrc`：包含Quartus工程以及全部Verilog源代码和测试代码
  - `LLE.slx`：Simulink模型，可以直接用MATLAB打开，需要安装Vision HDL Toolbox
